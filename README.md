# Code for the study of odd-parity altermagnetism
This repository stores the computation and analysis codes used for the research of odd-parity altermagnetism in the paper [arXiv:2503.09602](https://arxiv.org/abs/2503.09602). The setup procedure for the usage is as follows:
1. Clone this repo.
2. Download the [dataset](https://doi.org/10.34810/DATA3497) and put it into the repo directory.
3. Clone the [CMTKit](https://github.com/kappaping/cmtkit) repo under the same parent directory as this repo. Install the dependencies.
4. Create figs/ and data/ child directories within this repo directory.
5. Ready for use.
